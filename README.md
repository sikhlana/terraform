# Example IaaC

---

This is an example repository to showcase what you can do with [sikhlana/cdktfs](https://github.com/sikhlana/cdktfs).

### Tree-view

```
(App)
├─┬ infrastructure (Infrastructure)
│ ├── vars (Variables)
│ ├── allowed-ips (TerraformVariable)
│ ├── tls (TlsProvider)
│ ├── digitalocean (DigitaloceanProvider)
│ ├── region (DataDigitaloceanRegion)
│ ├── image (DataDigitaloceanImage)
│ ├─┬ mail (Mail)
│ │ ├── vpc (Vpc)
│ │ ├── private-key (PrivateKey)
│ │ ├── ssh-key (SshKey)
│ │ ├── droplet (Droplet)
│ │ └── firewall (Firewall)
│ ├─┬ web (Web)
│ │ ├── vpc (Vpc)
│ │ ├── private-key (PrivateKey)
│ │ ├── ssh-key (SshKey)
│ │ ├── droplet (Droplet)
│ │ └── firewall (Firewall)
│ └── backend (S3Backend)
├─┬ saifmahmud-name (SaifmahmudName)
│ ├── namedotcom (NamedotcomProvider)
│ ├── cloudflare (CloudflareProvider)
│ ├── zone (DataCloudflareZone)
│ ├── ns (DomainNameservers)
│ ├─┬ mail (Construct)
│ │ └─┬ apex (Construct)
│ │   ├── a (Record)
│ │   └── aaaa (Record)
│ ├─┬ apex (Construct)
│ │ ├─┬ apex (Construct)
│ │ │ ├── a (Record)
│ │ │ └── aaaa (Record)
│ │ └─┬ www (Construct)
│ │   ├── a (Record)
│ │   └── aaaa (Record)
│ └── backend (LocalBackend)
└─┬ gfnlabs-com (GfnlabsCom)
  ├── namecheap (NamecheapProvider)
  ├── cloudflare (CloudflareProvider)
  ├── zone (DataCloudflareZone)
  ├── ns (DomainRecords)
  ├─┬ apex (Construct)
  │ ├─┬ apex (Construct)
  │ │ ├── a (Record)
  │ │ └── aaaa (Record)
  │ └─┬ www (Construct)
  │   ├── a (Record)
  │   └── aaaa (Record)
  └── backend (LocalBackend)
```
